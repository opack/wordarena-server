Installation de wordarenad:
/!\ ATTENTION ! Le fichier doit être au format linux, donc attention aux retours à la ligne (juste LF, pas CRLF !)
	- S'assurer qu'un utilisateur existe pour lancer cet exécutable
		useradd wordarena
	- Rendre wordarenad exécutable
		chmod +x /etc/init.d/wordarenad
	- Le copier dans /etc/init.d/
		cp wordarenad /etc/init.d/
	- Tester le service
		sudo service wordarenad start
		sudo service wordarenad stop
		sudo service wordarenad restart
